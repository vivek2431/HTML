# HTML Image Tag

The image tag, `<img>`, is used to embed images into HTML documents. It is a self-closing tag and does not have a closing tag.

## Syntax:

```html
<img src="image.jpg" alt="Image Description">

# Image Tag Attributes

The `<img>` tag in HTML is used to embed images into web pages. It supports several attributes to control various aspects of the image presentation and behavior:

- `src`: Specifies the URL or file path of the image to be displayed.
  Example: `<img src="image.jpg" alt="Image Description">`

- `alt`: Specifies an alternate text for the image. It is displayed if the image cannot be loaded or by screen readers for accessibility purposes.
  Example: `<img src="image.jpg" alt="Alternate Text">`

- `width`, `height`: Specifies the width and height of the image, respectively, in pixels.
  Example: `<img src="image.jpg" alt="Image" width="100" height="100">`

- `title`: Provides additional information about the image, often displayed as a tooltip when the user hovers over the image.
  Example: `<img src="image.jpg" alt="Image" title="Image Title">`

- `loading`: Specifies how the image should be loaded. Possible values include lazy, eager, and auto.
  Example: `<img src="image.jpg" alt="Image" loading="lazy">`

The image tag is essential for adding visual content to web pages, enhancing user experience, and conveying information effectively.
